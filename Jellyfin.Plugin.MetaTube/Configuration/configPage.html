MetaTube

MetaTube 插件（适用于 Jellyfin/Emby）。 了解更多.

通用

MetaTube 服务端完整地址，建议使用 HTTPS。
MetaTube 服务端访问令牌；如果后端未设置令牌可留空。
按系列自动创建合集。
将导演写入对应视频元数据。
显示来源站点的社区评分。
生成在线视频预告（strm 格式）。
从 AVBASE 获取并替换为演员真名。

角标

为主图添加中文字幕角标。
自定义角标地址，建议使用 PNG。（默认：zimu.png）

图片

主图宽高比，设为负数时使用默认值。
JPEG 图片默认压缩质量，范围 0 到 100。（默认：90）

数据源

过滤并重排影片来源的搜索结果。
来源名称不区分大小写，按从左到右优先级递减，使用逗号分隔。

模板

预定义模板变量可参考 这里.

翻译

自定义 Google 翻译 API 地址。（可选）
自定义兼容 DeepL 的 API 地址。（可选）
自定义兼容 OpenAI 的 API 地址。（可选）
自定义兼容 OpenAI 的模型名。（可选）

替换

每行一条记录，使用等号分隔。目标文本留空表示删除源文本。
每行一条记录，使用等号分隔。目标演员留空表示删除源演员。
每行一条记录，使用等号分隔。目标类型留空表示删除源类型。
